JAVA의 장단점 및 사용 이유

- 장점
자바는 일반적으로 운영체제 위에서 실행되는 다른 언어의 프로그램들과는 달리
JVM위에서 동작한다. 특정 플랫폼에 종속적이지 않고 독립적이라는 장점이 있다.
운영체제위 JVM이 동작하고 JVM이 자바 프로그램의 운영체제 역할을 하게 된다.

다른 언어와 비교하여 메모리 관리가 쉽다.
자바의 Garbage Collector가 자동으로 메모리 관리를 해주기때문에 일반적으로 프로그래머는 상대적으로 메모리 관리에서
자유롭다

시스템에 상관없이 멀티스레드를 구현할 수 있다.

별도의 얘기로 한국에서 사용시 한국어로 변역된 자료나 레퍼런스를 찾기 쉽다.

- 단점
대체적으로 코드의 길이가 길다
ex) system.out.println
    
다른언어에 비해 상대적으로 속도가 느리다
최근 JIT컴파일러 개선과 신기술 도입으로 JVM 기능이 향상됨으로 속도문제가 상당히 개선된걸로 알고 있다.
